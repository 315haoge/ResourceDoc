# 第三方库 - Swift（2.0+）
## 网络-Internet
- [AEXML][1]：XML解析器，Simple and lightweight XML parser written in Swift
- [Alamofire][2]：**HTTP网络库，AFNetworking作者写的swift版本**，Elegant HTTP Networking in Swift。参考文：
	1. [swift网络库Alamofire源码分析][3]
	2. [Alamofire网络库基础教程][4]
	3. [Alamofire网络库进阶教程][5]
	4. [Alamofire - 优雅的处理 Swift 中的网络操作][6]
	5. [如何在 Swift 中使用 Alamofire 进行网络编程][7]
- [blackfish][8]：基于 Node.js/Express 风格的 Web 服务器，A fast HTTP web server based on Node.js and Express written in Swift
- [CaesarParser][9]：JSON 到 Model 转换工具，通过自定义操作符简化了代码编写，CaesarParser is a framework written in Swift for you to parse Model from JSON or to JSON.
- [Decodable][10]：**JSON解析，Swift 2 JSON parsing done (more) right**
- [Freddy][11]：更安全地解析，且充分考虑 Swift 固有语法的 JSON 解析类。A reusable framework for parsing JSON in Swift.
- [Genome][12]：A simple, type safe, failure driven mapping library for serializing JSON to models in Swift 2.0
- [HanekeSwift][13]：网络数据缓存库，A lightweight generic cache for iOS written in Swift with extra love for images.
- [Kingfisher][14]：A lightweight and pure Swift implemented library for downloading and caching image from the web.
- [Luminescence][15]：HTTP parser for Swift 2
- [netfox][16]：A lightweight, one line setup, iOS network debugging library!
- [Pitaya][17]：Pitaya(火龙果) 是一个写给人看的纯 Swift 写成的 HTTP / HTTPS 网络库。
- [Perfect][18]：用 Swift 实现的应用服务器，自带 Web 开发框架并支持 REST
- [Redes][19]：High-level network layer abstraction library written in Swift.
- [Reachability.swift][20]：重写reachability网络库，用于替换苹果的 Reachability 类，可以方便地检测当前是否联网以及具体的联网状态，Replacement for Apple's Reachability re-written in Swift with closures
- [RxGitHubAPI][21]：一个用RxSwift实现的GitHubAPI库，参考：
	1. [A GitHub API by RxSwift][22]
	2. [Swift，RxSwift实现的RxGithub API库用法和代码说明][23]
- [Swift JSON mapping libraries][24]，几个Swift使用的JSON解析库试用说明，包括[ObjectMapper][25]，[Genome][26]，[Argo][27]，[Decodable][28]，和[Unbox][29]
- [SPTDataLoader][30]：Spotify官方开源的iOS端HTTP库，The HTTP library used by the Spotify iOS client
- [swifter][31]：Tiny http server engine written in Swift programming language.
- [SwiftyJSON][32]：JSON解析，The better way to deal with JSON data in Swift
- [Swallow][33]：Http服务器，封装了 HTTP 和 HTTPS 请求，支持链式操作和 JSON 解析，能满足绝大部分通信需求
- [siesta][34]：REST客户端框架
- [Venice][35]：高并发通信处理，CSP for Swift 2 (Linux ready)

- [Localize-Swift][39]：国际化及本地化字符串框架支持类库。Swift 2.0 friendly localization and i18n with in-app language switching

- [SwiftyText][41]：基于TextKit的文本渲染和手势处理库，SwiftyText in Swift based on Text Kit
- [TextFieldEffects][42]:Custom UITextFields effects inspired by Codrops
- [TTGTagCollectionView][43]：显示一些“标签”样式的内容，教程：[开源项目-TTGTagCollectionView][44]
