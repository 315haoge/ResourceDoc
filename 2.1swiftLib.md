# 第三方库 - Swift（2.0+）
## 网络-Internet
- [AEXML][1]：XML解析器，Simple and lightweight XML parser written in Swift
- [Alamofire][2]：**HTTP网络库，AFNetworking作者写的swift版本**，Elegant HTTP Networking in Swift。参考文：
	1. [swift网络库Alamofire源码分析][3]
	2. [Alamofire网络库基础教程][4]
	3. [Alamofire网络库进阶教程][5]
	4. [Alamofire - 优雅的处理 Swift 中的网络操作][6]
	5. [如何在 Swift 中使用 Alamofire 进行网络编程][7]
- [CaesarParser][8]：JSON 到 Model 转换工具，通过自定义操作符简化了代码编写，CaesarParser is a framework written in Swift for you to parse Model from JSON or to JSON.
- [Decodable][9]：**JSON解析，Swift 2 JSON parsing done (more) right**
- [Genome][10]：A simple, type safe, failure driven mapping library for serializing JSON to models in Swift 2.0
- [HanekeSwift][11]：网络数据缓存库，A lightweight generic cache for iOS written in Swift with extra love for images.
- [Kingfisher][12]：A lightweight and pure Swift implemented library for downloading and caching image from the web.
- [Luminescence][13]：HTTP parser for Swift 2
- [netfox][14]：A lightweight, one line setup, iOS network debugging library!
- [Pitaya][15]：Pitaya(火龙果) 是一个写给人看的纯 Swift 写成的 HTTP / HTTPS 网络库。
- [Perfect][16]：用 Swift 实现的应用服务器，自带 Web 开发框架并支持 REST
- [Redes][17]：High-level network layer abstraction library written in Swift.
- [Reachability.swift][18]：重写reachability网络库，用于替换苹果的 Reachability 类，可以方便地检测当前是否联网以及具体的联网状态，Replacement for Apple's Reachability re-written in Swift with closures
- [Swift JSON mapping libraries][19]，几个Swift使用的JSON解析库试用说明，包括[ObjectMapper][20]，[Genome][21]，[Argo][22]，[Decodable][23]，和[Unbox][24]
- [swifter][25]：Tiny http server engine written in Swift programming language.
- [SwiftyJSON][26]：JSON解析，The better way to deal with JSON data in Swift
- [Swallow][27]：Http服务器，封装了 HTTP 和 HTTPS 请求，支持链式操作和 JSON 解析，能满足绝大部分通信需求
- [siesta][28]：REST客户端框架
- [Venice][29]：高并发通信处理，CSP for Swift 2 (Linux ready)


## 文本工具
- [AutocompleteField][30]：自动补全工具，Add word completion to your UITextFields
- [FontBlaster][31]：以编程的方式，将自定义字体加入你的app
- [RichTextView][32]：使用 TextKit 定制全功能的 TextView
- [SwiftyText][33]：基于TextKit的文本渲染和手势处理库，SwiftyText in Swift based on Text Kit
- [TextFieldEffects][34]:Custom UITextFields effects inspired by Codrops
- [TTGTagCollectionView][35]：显示一些“标签”样式的内容，教程：[开源项目-TTGTagCollectionView][36]


## 动效和动画
- [ios-charts][37]：画图标，Beautiful charts for iOS apps!
- [MaterialKit][38]：显示特效框架，A beautiful Material Design framework in Swift. 
- [MPParallaxView][39]：3D图像动态效果
- [NVActivityIndicatorView][40]：一个加载动画的合集库
