# 第三方库 - Swift（2.0+）
## 网络-Internet
- [AEXML][1]：XML解析器，Simple and lightweight XML parser written in Swift
- [Alamofire][2]：**HTTP网络库，AFNetworking作者写的swift版本**，Elegant HTTP Networking in Swift。参考文：
	1. [swift网络库Alamofire源码分析][3]
	2. [Alamofire网络库基础教程][4]
	3. [Alamofire网络库进阶教程][5]
	4. [Alamofire - 优雅的处理 Swift 中的网络操作][6]
	5. [如何在 Swift 中使用 Alamofire 进行网络编程][7]
- [CaesarParser][8]：JSON 到 Model 转换工具，通过自定义操作符简化了代码编写，CaesarParser is a framework written in Swift for you to parse Model from JSON or to JSON.
- [Decodable][9]：**JSON解析，Swift 2 JSON parsing done (more) right**
- [Genome][10]：A simple, type safe, failure driven mapping library for serializing JSON to models in Swift 2.0
- [HanekeSwift][11]：网络数据缓存库，A lightweight generic cache for iOS written in Swift with extra love for images.
- [Luminescence][12]：HTTP parser for Swift 2
- [netfox][13]：A lightweight, one line setup, iOS network debugging library!
- [Pitaya][14]：Pitaya(火龙果) 是一个写给人看的纯 Swift 写成的 HTTP / HTTPS 网络库。
- [Perfect][15]：用 Swift 实现的应用服务器，自带 Web 开发框架并支持 REST
- [Redes][16]：High-level network layer abstraction library written in Swift.
- [Reachability.swift][17]：重写reachability网络库，用于替换苹果的 Reachability 类，可以方便地检测当前是否联网以及具体的联网状态，Replacement for Apple's Reachability re-written in Swift with closures
- [Swift JSON mapping libraries][18]，几个Swift使用的JSON解析库试用说明，包括[ObjectMapper][19]，[Genome][20]，[Argo][21]，[Decodable][22]，和[Unbox][23]
- [swifter][24]：Tiny http server engine written in Swift programming language.
- [SwiftyJSON][25]：JSON解析，The better way to deal with JSON data in Swift
- [Swallow][26]：Http服务器，封装了 HTTP 和 HTTPS 请求，支持链式操作和 JSON 解析，能满足绝大部分通信需求
- [siesta][27]：REST客户端框架
- [Venice][28]：高并发通信处理，CSP for Swift 2 (Linux ready)


## 文本工具
- [AutocompleteField][29]：自动补全工具，Add word completion to your UITextFields
- [FontBlaster][30]：以编程的方式，将自定义字体加入你的app
- [RichTextView][31]：使用 TextKit 定制全功能的 TextView
- [SwiftyText][32]：基于TextKit的文本渲染和手势处理库，SwiftyText in Swift based on Text Kit
- [TextFieldEffects][33]:Custom UITextFields effects inspired by Codrops
- [TTGTagCollectionView][34]：显示一些“标签”样式的内容，教程：[开源项目-TTGTagCollectionView][35]


## 动效和动画
- [ios-charts][36]：画图标，Beautiful charts for iOS apps!
- [MaterialKit][37]：显示特效框架，A beautiful Material Design framework in Swift. 
- [MPParallaxView][38]：3D图像动态效果
- [NVActivityIndicatorView][39]：一个加载动画的合集库
- [Spring][40]：iOS动画框架
- [Splitflap][41]：一个翻牌子的特效，A simple split-flap display for your Swift applications
- [StarWars.iOS][42]：炫酷的星战动画效果，This component implements transition animation to crumble view-controller into tiny pieces.

## 绘制、显示组件
- [Cartography][43]：适配工具，A declarative Auto Layout DSL for Swift
- [ESTCollectionViewDropDownList][44]：一个实现了 iOS 下拉标签列表视图的 Demo，创作灵感来自于虾米
- [ESTMusicIndicator][45]：音乐播放的动态效果控件
- [ENSwiftSideMenu][46]：一个像 Gmail 客户端那样的侧滑菜单，参考：[在 Swift 项目中实现侧滑菜单-利用 SWRevealViewController][47]
- [ElasticTransition][48]：皮筋侧滑效果
- [ImageViewer][49]：图片全屏预览组件及类库
- [IBAnimatable][50]：基于 Xcode 插件技术快速原型开发工具。Design and prototype UI, interaction, navigation, transition and animation for App Store ready Apps in Interface Builder with IBAnimatable.
- [PNChart-Swift][51]：画图表工具，A simple and beautiful chart lib used in Piner and CoinsMan for iOS
- [RangeSlider][52]：数值范围选择器
- [SnapKit][53]：**A Swift Autolayout DSL for iOS & OS X**
- [UIImageView-BetterFace-Swift][54]：更好显示人脸的缩略图工具
- [Whisper][55]：用三种不同的样式来展示应用内通知，给用户带来一些新意。Break the silence of your UI, whispering, shouting or whistling at it. 
- [WaveLoadingView][56]：波浪进度指示器，参考：[就叫波浪进度指示器吧WaveLoadingIndicator][57]

- [C4iOS][60]：用更少的代码，创建最佳应用体验库。C4 is an open-source creative coding framework that harnesses the power of native iOS programming with a simplified API that gets you working with media right away. Build artworks, design interfaces and explore new possibilities working with media and interaction.

- [Carlos][62]：缓存框架，A simple but flexible cache
- [Interstellar][63]：轻量级 Functional Reactive Programming 库
- [MockFive][64]：在 TDD 开发中，需要先跑通测试然后再写具体实现，因此需要先伪造出实现类和函数来通过测试，这就是 Mock。A Mocking Framework for Swift Unit Tests

## 数据处理
- [ALBNoSQLDB][78]：SQLite封装，A SQLite database wrapper written in Swift that requires no SQL knowledge to use
- [AWSQLiteDB][79]：AWSQLiteDB a simple wrapper for libsqlite3.0 in Swift
- [GRDB.swift][80]：SQL数据库工具集，A versatile SQLite toolkit for Swift
- [Pantry][81]：轻量级数据持久化库，Pantry is a lightweight way to persist structs containing user data, cached content or other relevant objects for later retrieval.
- [RealmSwift-swift2.0][82]：**流行的小型数据库，core data替代品**，Realm is a mobile database: a replacement for Core Data & SQLite。中文教程：[使用 Realm 和 Swift 创建 ToDo 应用][83]，以及范例文章：[iOS：选择 Realm 而不是 CoreData][84]
- [Regex][85]：正则表达式工具
- [SQLite.swift][86]：纯 Swift 实现的类型安全的 SQLite3 封装，A type-safe, Swift-language layer over SQLite3.
- [Swift-Flow][87]：数据流工具，单向数据流可以简化代码逻辑和依赖关系，让代码更健壮。Unidirectional Data Flow in Swift
- [SwiftVerbalExpressions][88]：正则表达式库，SwiftVerbalExpressions is a Swift library that helps to construct difficult regular expressions - ported from the awesome JavaScript VerbalExpressions.
