# Swift第三方库 - 绘制、显示用组件
## 特效
- [ConfettiView][1]：五彩碎纸雨效果，Confetti View lets you create a magnificent confetti view in your app
- [ElasticTransition][2]：皮筋侧滑效果
- [expanding-collection][3]：展开、查看列表项的 Peek/Pop 用法及动画效果，ExpandingCollection is a card peek/pop controller
- [FlowingMenu][4]：皮筋弹跳式菜单出场，Interactive view transition to display menus with flowing and bouncing effects in Swift
- [folding-cell][5]：自然流畅、清新的单元格折叠动画类及演示，FoldingCell is an expanding content cell inspired by folding paper material
- [RevealingSplashView][6]：图标缩放、背景、颜色可定制的动画转场效果（Swift），A Splash view that animates and reveals its content, inspired by Twitter splash
- [RippleEffectView][7]：加载时实现与背景的涟漪效应（连锁反应），RippleEffectView - A Neat Rippling View Effect
- [Splitflap][8]：一个翻牌子的特效，A simple split-flap display for your Swift applications 
- [SpreadButton][9]：一个当你点击它后会像花或者镰刀一样展开的按钮
- [TableFlip][10]：简单、实用、酷炫地 UITableView 出场动画，A simpler way to do cool UITableView animations!
- [UPCarouselFlowLayout][11]：和图片轮播类似的走马灯效果，当前显示的内容会被放大，手指滑动就可以左右切换内容。A fancy carousel flow layout for UICollectionView on iOS.
- [Whisper][12]：用三种不同的样式来展示应用内通知，给用户带来一些新意。Break the silence of your UI, whispering, shouting or whistling at it. 
- [WaveLoadingView][13]：波浪进度指示器，参考：[就叫波浪进度指示器吧WaveLoadingIndicator][14]

## 扩展控件、框架
- [ActionSheetController][15]：类似 UIAlertController API 风格的 Action Sheet弹窗
- [AppStoreStyleHorizontalScrollView][16]：仿 AppStore 应用的水平滚动视图演示，App store style horizontal scroll view
- [AspectFillFaceAware][17]：让脸部恰当地显示于视图中心位置的UIImageView 扩展，An extension that gives UIImageView the ability to focus on faces within an image when using AspectFill.
- [AudioIndicatorBars][18]：同时支持代码及 IB 方式的音频播放指示条组件，AIB indicates for your app users which audio is playing. Just like the Podcasts app.
- [BouncyPageViewController][19]：反弹效果的页面视图控件，Page view controller with bounce effect
- [BTNavigationDropdownMenu][20]：用Swift实现的下拉框菜单第三方框架，The elegant dropdown menu, written in Swift, appears underneath navigation bar to display a list of related items when a user click on the navigation title.
- [CardSlider][21]：Tinder风格的卡片视图，Innovative twist to Tinder cards for iOS.
- [CascadingTableDelegate][22]：UITableView 内对应复杂 DataSource（富文本内容）展现框架，A no-nonsense way to write cleaner UITableViewDelegate and UITableViewDataSource in Swift.
- [CBMDTabbarController][23]：Material Design 风格导航栏 iOS 实现，It is a smooth MD tabbarController used on iOS, which implement by Swift.
- [ColorArcProgressBar][24]：可定制的圆形进度条，通过xml参数配置可实现QQ健康中步数的弧形进度显示、仪盘表显示速度、最常见的下载进度条等功能
- [CRNetworkButton][25]：好玩、有新意（用于网络或服务器请求）的交互按钮
- [DMSwipeCards][26]：类似Tinder的卡片界面
- [DropDown][27]：一个符合 Material Design 的下拉菜单，支持自定义，可以实现输入+选择这种组合控件。A Material Design drop down for iOS
- [ELWaterFallLayout][28]：自定义瀑布流布局 UI 组件
- [ENSwiftSideMenu][29]：一个像 Gmail 客户端那样的侧滑菜单，参考：[在 Swift 项目中实现侧滑菜单-利用 SWRevealViewController][30]
- [ESTMusicIndicator][31]：音乐播放的动态效果控件
- [Gecco][32]：一个高亮引导向导界面库，Simply highlight items for your tutorial walkthrough
- [iOS-MultiSelectionTable][33]：iOS中多选表格的漂亮方法。Beautiful way of having a multi-selection table on iOS written in Swift
- [JFCardSelectionViewController][34]：罗盘式照片预览、选取视图控制器组件，A fancy collection style view controller
- [JRAlertController][35]：基于apple的UIAlertController控件api，用swift重新打造的UI控件，更符合主流app的风格。参考：[JRAlertController][36]
- [JTAppleCalendar][37]：功能强大、实用的日历组件，The Unofficial Swift Apple Calendar Library. View. Control. for iOS & tvOS
- [Kaeru][38]：仿 iOS 任务切换管理的 NavigationController 扩展实现，Switch viewcontroller like ios task manager
- [Koyomi][39]：可定制日历，Simple customizable calendar component in Swift
- [KYShutterButton][40]：模拟了 iOS 相机按钮，包括慢动作和延时效果，可以自定义颜色。KYShutterButton is a custom button that is similar to the shutter button of the camera app
- [Material][41]：能很好地帮助你迅速打造一款在 iOS 上的 Material 风格应用，它提供了像是按钮、导航栏、点击的动画效果等一系列 Material 风格的 UI 元素供你使用。A beautiful graphics framework for Material Design in Swift. 
- [MRefresh][42]：可自定义的下拉刷新控件，This pod enables you to add pull-to-refresh mechanism to your scrollviews and tableviews and use svg patterns in your refreshing view's layers.
- [NextGrowingTextView][43]：自适应高度的多行文本输入框，The next in the generations of 'growing textviews' optimized for iOS 7 and above.
- [NoticeBar][44]：iOS 仿 QQ 与 pin 的消息提示栏，可自定义样式，A simple NoticeBar written by Swift 3, similar with QQ notice view.
- [PageControls][45]：定制的页面控制组件，可替换 UIPageControl，This is a selection of custom page controls to replace UIPageControl
- [PinterestSegment][46]：带有遮罩动画的类似Pinterest的段落控制器，A Pinterest-like segment control with masking animation. 
- [PMAlertController][47]：可定制弹窗组件替代官版不可定制的 UIAlertController。PMAlertController is a great and customizable substitute to UIAlertController
- [Popover][48] ：UIPopoverController 效果的 Swift 实现，类似微信首页右上角按钮的弹出视图。Popover is an UIPopoverController like control for iOS.
- [PopupDialog][49]：简洁、可定制性高的弹出框，用于取代UIAlertControllers 的默认样式，A simple, customizable popup dialog for iOS written in Swift. Replaces UIAlertControllers alert style.
- [pull-to-refresh][50]：一个非常易于开发者使用的下拉刷新和加载更多组件。通过一个UIScrollView的扩展，可以轻松为UIScrollView的所有子类添加下拉刷新功能。 如果你想定制组件的UI样式，只要实现指定的协议方法即可。
- [Presentr][51]：正确姿势封装最常见模态视图，使之更易用，Swift wrapper for custom ViewController presentations on iOS
- [RangeSlider][52]：数值范围选择器
- [Reactions][53]：仿Facebook点赞评价功能，Fully customizable Facebook reactions control
- [Render][54]：启发自 React 的纯 Swift 函数版基于 UIKit 封装类库，Swift and UIKit a la React.
- [RHPreviewCell][55]：遵从简洁交互设计，通过遮罩层快速预览单元格内容
- [Segmentio][56]：功能完善的带动画的可定制分段式控制组件，Animated top/bottom segmented control written in Swift. 
- [SideMenuController][57]：侧边栏控制器，支持自定义位置、配置动画效果和自定义状态栏行为。A side menu controller written in Swift
- [SJFluidSegmentedControl][58]：一个自定义的具有交互转换的分段控件，A segmented control with custom appearance and interactive animations. Written in Swift 3.0.
- [SkyFloatingLabelTextField][59]：带图标显示浮动标签组件库，A beautiful and flexible text field control implementation of "Float Label Pattern". Written in Swift. As used in the Skyscanner TravelPro iOS app.
- [Smile-Lock][60]：可定制密码锁组件，A library for make a beautiful Passcode Lock View
- [SnapchatCheckbox][61]：使一个小盒子以圆形的旋转方式被填充的仿snapchat复选框，A Snapchat-inspired checkbox.
- [SnapTimer][62]：定制 UIView 模仿 Snapchat's stories 定时器效果
- [SubmitButton][63]：带进度条动画的提交按钮，An animated Submit Button written in Swift.
- [SubtleVolume][64]：更隐蔽的系统音量替代指示器，Replace the system volume popup with a more subtle indicator.
- [SwipeViewController][65]：页面滑动和标签选项卡，SwipeViewController is a Swift modification of RKSwipeBetweenViewControllers - navigate between pages / ViewControllers
- [Swift-CircleMenu][66]：旋转环形菜单，Rotating circle menu written in Swift 3 
- [SwiftMessages][67]：高可定制信息弹窗组件库，A very flexible message bar for iOS written in Swift.
- [StackViewController][68]：方便 iOS 开发者使用 UIStackView 构建表单或其它静态内容视图 。A controller that uses a UIStackView and view controller composition to display content in a list
- [Toast-Swift][69]：高可定制易用的 Toast 弹出通知组件，A Swift extension that adds toast notifications to the UIView object class.
- [TabDrawer][70]：可定制的TabBar，TabDrawer is a customizable TabBar UI element that allows you to run a block of code upon TabBarItem selection, or display a customizable drawer. 
- [TabPageViewController][71]：一款实用的分页视图及滚动选项卡组件，Paging view controller and scroll tab view
- [TimelineTableViewCell][72]：一个由UITableViewCell实现的时间线视图，Simple timeline view implemented by UITableViewCell written in Swift 3.0
- [ViewStyle][73]：提供了一种类似于CSS的方法，来设置iOS中视图的外观样式，希望将视图的结构与样式设置分离。ViewStyle provides a way to set UIView's visual appearance properties like CSS
- [WCLShineButton][74]：类似太阳动画的button，This is a UI lib for iOS. Effects like shining.
- [WHC\_PageViewKit][75]：iOS 平台轻量级的 PageView 组件，其中TitleBar拥有30多种UI样式
- [YNDropDownMenu][76]：下拉菜单扩展，Awesome Dropdown menu for iOS with Swift 3 
- [YPDropMenuViewDemo][77]：swift实现的一款DropMenu,menuView,类似之前美团的下拉菜单，支持swift2.2，因为用到iOS8.的毛玻璃。所以想支持到iOS8.0,支持自定义，支持等宽排列，支持从左向右排列
- [Yoshi][78]：完成度较高地可定制调试菜单组件封装，A convenient wrapper around the UI code that is often needed for displaying debug menus.
- [ZYThumbnailTableView][79]：可展开型预览TableView，a TableView have thumbnail cell only, and you can use gesture let it expands other expansionView, all diy。参考：[ZYThumbnailTableView---堪比一个小型阅读App][80]

- [Charts][83]：图表库，支持缩放、拖拽、坐标轴和图例自定义、动画以及其他功能。Beautiful charts for iOS/tvOS/OSX! The Apple side of the crossplatform MPAndroidChart.参考：[iOS 使用Charts框架 折线,柱状,K线,饼状,雷达全攻略][84]
- [CreditCardForm-iOS][85]：让开发人员创建仿实际信用卡UI。 CreditCardForm is iOS framework that allows developers to create the UI which replicates an actual Credit Card.
- [CTPanoramaView][86]：使用SceneKit来显示完整的球形或者圆柱形的全景触摸或者基于运动的控制。A library that displays spherical or cylindrical panoramas with touch or motion based controls.
- [DynamicButton][87]：带动画的特殊字符按钮显示，Yet another animated flat buttons in Swift 
- [Floater\_][88] ：加一个浮动指尖图标到你的演示应用，如何安装、使用步骤都写的比较清楚。Add a floating fingertip to your app demo.
- [ForceBlur][89]：采用 Force Touch + Blur 显示稳私图片信息，ForceBlur Animation for iOS Messaging Apps 
- [FSPagerView][90]：优雅的屏幕幻灯片库，主要使用UICollectionView实现。它有助于制作Banner，产品展示，欢迎/指引页，屏幕/ViewController滑块。FSPagerView is an elegant Screen Slide Library. It is extremely helpful for making Banner View、Product Show、Welcome/Guide Pages、Screen/ViewController Sliders.
- [Graphs][91]：非常易用的轻量级图表绘制库，Light weight charts view generater for iOS. Written in Swift.
- [greedo-layout-for-ios][92]：响应式网格布局，能够在固定 cell 高度的情况下自动调整图片宽度，让每行的图片都能以最佳状态显示。Full aspect ratio grid layout for iOS
- [Hexacon][93]：在手机上用多边形展示数据，类似 Apple Watch，A fancy hexagonal layout for displaying data like your Apple Watch
- [IoniconsKit][94]：让 UIImage / UIFont 更方便地使用自定义图标，Use ionicons as UIImage / UIFont in your projects with Swifty manners. 
- [ISTimeline][95]：一个简单的时间线界面，每个时间点可以自定义标题、描述、颜色，还支持触摸触发回调。Simple timeline view written in Swift 2.2
- [Macaw][96]：支持SVG的向量图形库，Powerful and easy-to-use vector graphics Swift library with SVG support
- [MFCard][97]：独特的信用卡输入验证控件，Easily integrate Credit Card payments in iOS App. 
- [NumberMorphView][98]：可爱的数字补间（变身）动画类库，A label view for displaying numbers which can transition or animate using a technique called number tweening or number morphing
- [OKKLineSwift][99]：用Swift3实现的绘制股票K线库
- [PieCharts][100]：易于使用和高度可定制的饼图库，Easy to use and highly customizable pie charts library for iOS
- [PNChart-Swift][101]：画图表工具，A simple and beautiful chart lib used in Piner and CoinsMan for iOS
- [preview-transition][102]：图片展示预览和转场，PreviewTransition is a simple preview gallery controller
- [Scrollable-GraphView][103]：灵动感十足的自适应、可定制滚动曲（折）线图表库，An adaptive scrollable graph view for iOS to visualise simple discrete datasets. 
- [Snowflake][104]：SVG绘图，SVG in Swift
- [StatefulViewController][105]：基于内容状态（加载中、错误、空）的占位符实用视图组件，Placeholder views based on content, loading, error or empty states
- [SwiftLinkPreview][106]：生成 URL预览，并且根据自己的喜好显示不同内容，比如图片、标题、文本等等。It makes a preview from an url, grabbing all the information such as title, relevant texts and images. 
- [TimelineTableViewCell][107]：简易时间轴视图组件，Simple timeline view implemented by UITableViewCell written in Swift 3.0
- [TTGSnackbar][108]：Android 下 SnackBar 的 iOS 版实现，TTGSnackbar shows simple message and action button on the bottom or top of the screen with multi kinds of animation, which is written in Swift3 and inspired by Snackbar in Android. It also support showing custom view, icon image or multi action button.
