# Swift第三方库 - UI
## 动效和动画
- [Advance][1]：侧重在手势交互、帧动画、自定义动画及仿真类型的动画框架库，A powerful animation framework for iOS and OS X.
- [D3View][2]：[一句代码轻松实现IOS的常用动画效果-Swift][3]
- [EasyAnimation][4]：A Swift library to take the power of UIView.animateWithDuration(\_:, animations:...) to a whole new level - layers, springs, chain-able animations and mixing view and layer animations together!
- [FlightAnimator][5]：基于 CoreAnimation 构建的动画库，使用 block 可以轻松创建动画。支持 43 种动画曲线，支持串联动画，极其强大。Natural Blocks Based Core Animation Framework
- [IBAnimatable][6]：基于 Xcode 插件技术快速原型开发工具。Design and prototype UI, interaction, navigation, transition and animation for App Store ready Apps in Interface Builder with IBAnimatable.参考教程：[iOS主流动画开源库深入分析：IBAnimatable][7]
- [Interpolate][8]：给各种手势加上默认的动画效果，也可以自定义动画。Swift interpolation for gesture-driven animations
- [ios-charts][9]：画图标，Beautiful charts for iOS apps!
- [Material][10]：显示特效框架，A beautiful Material Design framework in Swift. 
- [MotionMachine][11]：功能强大、优雅、模块化动画库。A powerful, elegant, and modular animation library for Swift.
- [MPParallaxView][12]：3D图像动态效果
- [NVActivityIndicatorView][13]：一个加载动画的合集库
- [ParticlesLoadingView][14]：通过 SpriteKit 内置工具粒子发射器实现酷炫的可定制装载动画，A customizable SpriteKit particles animation on the border of a view.
- [Spring][15]：iOS动画框架
- [Splitflap][16]：一个翻牌子的特效，A simple split-flap display for your Swift applications
- [StarWars.iOS][17]：炫酷的星战动画效果，This component implements transition animation to crumble view-controller into tiny pieces.参考文：[Swift炫酷动效StarWars.iOS剖析][18]
- [Stellar][19]：通过物理引擎 UIDynamic 实现超酷物理动画类库，简化 UIKit Dynamics 的操作，轻松实现各种动画效果。A fantastic Physical animation library for swift
- [XLActionController][20]：具有丰富可定制风格及动效的 Action 控制类库，Fully customizable and extensible action sheet controller written in Swift 2

## 文本显示
- [AutocompleteField][21]：文本自动补全工具，Add word completion to your UITextFields
- [Euler][22]：定制操作符实现直观、简洁的数学表达式` ∛27÷3+∑[3,1,2]` 类库，Swift Custom Operators for Mathematical Notation
- [FontBlaster][23]：以编程的方式，将自定义字体加入你的app
- [GlitchLabel][24]：一种闪瞎眼的字体渲染样式
- [Markingbird][25]：Markdown 解析器，翻译自 MarkdownSharp，后者是 Stack Overflow 使用的 Markdown 解析器。Markdown processor written in Swift (translation of MarkdownSharp)
- [Marklight][26]：Markdown 语法高亮显示编辑库，Markdown syntax highlighter for iOS
- [RichTextView][27]：使用 TextKit 定制全功能的 TextView
- [SwiftyDown][28]：Swift 实现的 markdown 解析器，Simple, Elegant, Powerful markdown parser written in swift.
- [SwiftyText][29]：基于TextKit的文本渲染和手势处理库，SwiftyText in Swift based on Text Kit
- [SwiftyMarkdown][30]：Markdown解析器，可以将 Markdown 转换成 NSAttributedString
- [TextFieldEffects][31]:Custom UITextFields effects inspired by Codrops
- [TextAttributes][32]：An easier way to compose attributed strings
- [Translucid][33]：文字背景镂空特效，Lightweight library to set an Image as text background. 
- [TTGTagCollectionView][34]：显示一些“标签”样式的内容，教程：[开源项目-TTGTagCollectionView][35]
- [WSTagsField][36]：可定制、易用的标签文本框输入组件，An iOS text field that represents different Tags

## AutoLayout
- [EasyPeasy][37]：编程方式自动布局框架库，Auto Layout made easy
- [Cartography][38]：适配工具，A declarative Auto Layout DSL for Swift
 - [SnapKit][39]：**A Swift Autolayout DSL for iOS & OS X**，参考：[Swift Snapkit 看这篇基本就"够"了][40]
- [Stevia][41]：给人用的Auto layout库，Elegant view layout for iOS 

## 绘制、显示组件
- [ActionSheetController][42]：类似 UIAlertController API 风格的 Action Sheet弹窗
- [BTNavigationDropdownMenu][43]：用Swift实现的下拉框菜单第三方框架，The elegant dropdown menu, written in Swift, appears underneath navigation bar to display a list of related items when a user click on the navigation title.
- [Caishen][44]：信用卡支付控件，主要功能为显示UI和获取数据，非银行数据交互。A Payment Card UI & Validator for iOS
- [Charts][45]：图表库，支持缩放、拖拽、坐标轴和图例自定义、动画以及其他功能。Beautiful charts for iOS/tvOS/OSX! The Apple side of the crossplatform MPAndroidChart.参考：[iOS 使用Charts框架 折线,柱状,K线,饼状,雷达全攻略][46]
- [ColorArcProgressBar][47]：可定制的圆形进度条，通过xml参数配置可实现QQ健康中步数的弧形进度显示、仪盘表显示速度、最常见的下载进度条等功能
- [DisplaySwitcher][48]：不同布局之间的平滑切换，Custom transition between two collection view layouts
- [DropDown][49]：一个符合 Material Design 的下拉菜单，支持自定义，可以实现输入+选择这种组合控件。A Material Design drop down for iOS
- [Dwifft][50]：更加优雅的 reloadData 实现
- [DynamicButton][51]：带动画的特殊字符按钮显示，Yet another animated flat buttons in Swift 
- [ESTMusicIndicator][52]：音乐播放的动态效果控件
- [ENSwiftSideMenu][53]：一个像 Gmail 客户端那样的侧滑菜单，参考：[在 Swift 项目中实现侧滑菜单-利用 SWRevealViewController][54]
- [ElasticTransition][55]：皮筋侧滑效果
- [expanding-collection][56]：展开、查看列表项的 Peek/Pop 用法及动画效果，ExpandingCollection is a card peek/pop controller
- [Floater\_][57] ：加一个浮动指尖图标到你的演示应用，如何安装、使用步骤都写的比较清楚。Add a floating fingertip to your app demo.
- [FlowingMenu][58]：皮筋弹跳式菜单出场，Interactive view transition to display menus with flowing and bouncing effects in Swift
- [folding-cell][59]：自然流畅、清新的单元格折叠动画类及演示，FoldingCell is an expanding content cell inspired by folding paper material
- [Graphs][60]：非常易用的轻量级图表绘制库，Light weight charts view generater for iOS. Written in Swift.
- [greedo-layout-for-ios][61]：响应式网格布局，能够在固定 cell 高度的情况下自动调整图片宽度，让每行的图片都能以最佳状态显示。Full aspect ratio grid layout for iOS
- [Gecco][62]：一个高亮引导向导界面库，Simply highlight items for your tutorial walkthrough
- [Hexacon][63]：在手机上用多边形展示数据，类似 Apple Watch，A fancy hexagonal layout for displaying data like your Apple Watch
- [Iconic][64]：快速生成矢量图标，Auto-generated icon font library for iOS
- [ISTimeline][65]：一个简单的时间线界面，每个时间点可以自定义标题、描述、颜色，还支持触摸触发回调。Simple timeline view written in Swift 2.2
- [KYShutterButton][66]：模拟了 iOS 相机按钮，包括慢动作和延时效果，可以自定义颜色。KYShutterButton is a custom button that is similar to the shutter button of the camera app
- [Material][67]：能很好地帮助你迅速打造一款在 iOS 上的 Material 风格应用，它提供了像是按钮、导航栏、点击的动画效果等一系列 Material 风格的 UI 元素供你使用。A beautiful graphics framework for Material Design in Swift. 
- [NextGrowingTextView][68]：自适应高度的多行文本输入框，The next in the generations of 'growing textviews' optimized for iOS 7 and above.
- [NumberMorphView][69]：可爱的数字补间（变身）动画类库，A label view for displaying numbers which can transition or animate using a technique called number tweening or number morphing
- [PNChart-Swift][70]：画图表工具，A simple and beautiful chart lib used in Piner and CoinsMan for iOS
- [PMAlertController][71]：可定制弹窗组件替代官版不可定制的 UIAlertController。PMAlertController is a great and customizable substitute to UIAlertController
- [Popover][72] ：UIPopoverController 效果的 Swift 实现，类似微信首页右上角按钮的弹出视图。Popover is an UIPopoverController like control for iOS.
- [preview-transition][73]：图片展示预览和转场，PreviewTransition is a simple preview gallery controller
- [pull-to-refresh][74]：一个非常易于开发者使用的下拉刷新和加载更多组件。通过一个UIScrollView的扩展，可以轻松为UIScrollView的所有子类添加下拉刷新功能。 如果你想定制组件的UI样式，只要实现指定的协议方法即可。
- [RangeSlider][75]：数值范围选择器
- [Render][76]：启发自 React 的纯 Swift 函数版基于 UIKit 封装类库，Swift and UIKit a la React.
- [Scrollable-GraphView][77]：灵动感十足的自适应、可定制滚动曲（折）线图表库，An adaptive scrollable graph view for iOS to visualise simple discrete datasets. 
- [SideMenuController][78]：侧边栏控制器，支持自定义位置、配置动画效果和自定义状态栏行为。A side menu controller written in Swift
- [SkyFloatingLabelTextField][79]：带图标显示浮动标签组件库，A beautiful and flexible text field control implementation of "Float Label Pattern". Written in Swift. As used in the Skyscanner TravelPro iOS app.
- [Smile-Lock][80]：可定制密码锁组件，A library for make a beautiful Passcode Lock View
- [SpreadButton][81]：一个当你点击它后会像花或者镰刀一样展开的按钮
- [SubtleVolume][82]：更隐蔽的系统音量替代指示器，Replace the system volume popup with a more subtle indicator.
- [SwipeViewController][83]：页面滑动和标签选项卡，SwipeViewController is a Swift modification of RKSwipeBetweenViewControllers - navigate between pages / ViewControllers
- [SwiftLinkPreview][84]：生成 URL预览，并且根据自己的喜好显示不同内容，比如图片、标题、文本等等。It makes a preview from an url, grabbing all the information such as title, relevant texts and images. 
- [SwiftSVG][85]：实现高性能 SVG 解析，并且支持多种输入格式。A single pass SVG parser with multiple interface options (String, NS/UIBezierPath, CAShapeLayer, and NS/UIView)
- [StackViewController][86]：方便 iOS 开发者使用 UIStackView 构建表单或其它静态内容视图 。A controller that uses a UIStackView and view controller composition to display content in a list
- [Toast-Swift][87]：高可定制易用的 Toast 弹出通知组件，A Swift extension that adds toast notifications to the UIView object class.
- [TabDrawer][88]：可定制的TabBar，TabDrawer is a customizable TabBar UI element that allows you to run a block of code upon TabBarItem selection, or display a customizable drawer. 
- [UPCarouselFlowLayout][89]：和图片轮播类似的走马灯效果，当前显示的内容会被放大，手指滑动就可以左右切换内容。A fancy carousel flow layout for UICollectionView on iOS.
- [Whisper][90]：用三种不同的样式来展示应用内通知，给用户带来一些新意。Break the silence of your UI, whispering, shouting or whistling at it. 
- [WaveLoadingView][91]：波浪进度指示器，参考：[就叫波浪进度指示器吧WaveLoadingIndicator][92]
- [YPDropMenuViewDemo][93]：swift实现的一款DropMenu,menuView,类似之前美团的下拉菜单，支持swift2.2，因为用到iOS8.的毛玻璃。所以想支持到iOS8.0,支持自定义，支持等宽排列，支持从左向右排列
- [ZYThumbnailTableView][94]：可展开型预览TableView，a TableView have thumbnail cell only, and you can use gesture let it expands other expansionView, all diy。参考：[ZYThumbnailTableView---堪比一个小型阅读App][95]

## 图片显示及处理

- [CartoonEyes][97]：前置摄像头捕获图像后，采用 Core Image 脸部识别 CIDetector 和漫画效果滤镜复合出卡通效果眼睛。Composite Cartoon Eyes over Face from Front Camera with CoreImage
- [Fusuma][98]：类似Instagram风格的图片浏览及相机拍照功能库，Instagram-like photo browser and a camera feature with a few line of code in Swift.
- [Gifu][99]：GIF库，High-performance animated GIF support for iOS in Swift
- [GPUImage2][100]：一个GPU加速视频和图像处理Swift框架。GPUImage 2 is a BSD-licensed Swift framework for GPU-accelerated video and image processing.
- [ImageViewer][101]：图片全屏预览组件及类库
- [MetalAcc][102]：基于GPU加速图片处理，包括高性能的滤镜。GPU-based media processing library using Metal written in Swift
- [moa][103]：自动进行异步下载并显示图片的插件，An image download extension for image view written in Swift for iOS, tvOS and macOS.
- [LegofySwift][104]：将图片自动生成为 1x1 乐高积木拼接而成的图形框架库，Make images look as if they are made out of 1x1 LEGO blocks
- [PHImageKit][105]：图像下载&缓存，以及GIF显示库。Simple yet powerful image downloading, caching and GIF playback framework.
- [SKPhotoBrowser][106]：图片浏览工具，支持多种显示模式，支持缓存图片，支持缩放。Simple PhotoBrowser/Viewer inspired by facebook, twitter photo browsers written by swift
- [UIImageView-BetterFace-Swift][107]：更好显示人脸的缩略图工具
