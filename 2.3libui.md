# Swift第三方库 - UI
## 动效和动画
- [Advance][1]：侧重在手势交互、帧动画、自定义动画及仿真类型的动画框架库，A powerful animation framework for iOS and OS X.
- [D3View][2]：[一句代码轻松实现IOS的常用动画效果-Swift][3]
- [EasyAnimation][4]：A Swift library to take the power of UIView.animateWithDuration(\_:, animations:...) to a whole new level - layers, springs, chain-able animations and mixing view and layer animations together!
- [IBAnimatable][5]：基于 Xcode 插件技术快速原型开发工具。Design and prototype UI, interaction, navigation, transition and animation for App Store ready Apps in Interface Builder with IBAnimatable.参考教程：[iOS主流动画开源库深入分析：IBAnimatable][6]
- [Interpolate][7]：给各种手势加上默认的动画效果，也可以自定义动画。Swift interpolation for gesture-driven animations
- [ios-charts][8]：画图标，Beautiful charts for iOS apps!
- [Material][9]：显示特效框架，A beautiful Material Design framework in Swift. 
- [MotionMachine][10]：功能强大、优雅、模块化动画库。A powerful, elegant, and modular animation library for Swift.
- [MPParallaxView][11]：3D图像动态效果
- [NVActivityIndicatorView][12]：一个加载动画的合集库
- [ParticlesLoadingView][13]：通过 SpriteKit 内置工具粒子发射器实现酷炫的可定制装载动画，A customizable SpriteKit particles animation on the border of a view.
- [Spring][14]：iOS动画框架
- [Splitflap][15]：一个翻牌子的特效，A simple split-flap display for your Swift applications
- [StarWars.iOS][16]：炫酷的星战动画效果，This component implements transition animation to crumble view-controller into tiny pieces.
- [Stellar][17]：通过物理引擎 UIDynamic 实现超酷物理动画类库，简化 UIKit Dynamics 的操作，轻松实现各种动画效果。A fantastic Physical animation library for swift
- [XLActionController][18]：具有丰富可定制风格及动效的 Action 控制类库，Fully customizable and extensible action sheet controller written in Swift 2

## 文本显示
- [AutocompleteField][19]：文本自动补全工具，Add word completion to your UITextFields
- [Euler][20]：定制操作符实现直观、简洁的数学表达式` ∛27÷3+∑[3,1,2]` 类库，Swift Custom Operators for Mathematical Notation
- [FontBlaster][21]：以编程的方式，将自定义字体加入你的app
- [GlitchLabel][22]：一种闪瞎眼的字体渲染样式
- [Markingbird][23]：Markdown 解析器，翻译自 MarkdownSharp，后者是 Stack Overflow 使用的 Markdown 解析器。Markdown processor written in Swift (translation of MarkdownSharp)
- [Marklight][24]：Markdown 语法高亮显示编辑库，Markdown syntax highlighter for iOS
- [RichTextView][25]：使用 TextKit 定制全功能的 TextView
- [SwiftyDown][26]：Swift 实现的 markdown 解析器，Simple, Elegant, Powerful markdown parser written in swift.
- [SwiftyText][27]：基于TextKit的文本渲染和手势处理库，SwiftyText in Swift based on Text Kit
- [SwiftyMarkdown][28]：Markdown解析器，可以将 Markdown 转换成 NSAttributedString
- [TextFieldEffects][29]:Custom UITextFields effects inspired by Codrops
- [TextAttributes][30]：An easier way to compose attributed strings
- [TTGTagCollectionView][31]：显示一些“标签”样式的内容，教程：[开源项目-TTGTagCollectionView][32]

## AutoLayout
- [EasyPeasy][33]：编程方式自动布局框架库，Auto Layout made easy
- [Cartography][34]：适配工具，A declarative Auto Layout DSL for Swift
 - [SnapKit][35]：**A Swift Autolayout DSL for iOS & OS X**
- [Stevia][36]：给人用的Auto layout库，Elegant view layout for iOS 

## 绘制、显示组件
- [ActionSheetController][37]：类似 UIAlertController API 风格的 Action Sheet弹窗
- [BTNavigationDropdownMenu][38]：用Swift实现的下拉框菜单第三方框架，The elegant dropdown menu, written in Swift, appears underneath navigation bar to display a list of related items when a user click on the navigation title.
- [Caishen][39]：信用卡支付控件，主要功能为显示UI和获取数据，非银行数据交互。A Payment Card UI & Validator for iOS
- [Charts][40]：图表库，支持缩放、拖拽、坐标轴和图例自定义、动画以及其他功能。Beautiful charts for iOS/tvOS/OSX! The Apple side of the crossplatform MPAndroidChart.
- [ColorArcProgressBar][41]：可定制的圆形进度条，通过xml参数配置可实现QQ健康中步数的弧形进度显示、仪盘表显示速度、最常见的下载进度条等功能
- [DisplaySwitcher][42]：不同布局之间的平滑切换，Custom transition between two collection view layouts
- [Dwifft][43]：更加优雅的 reloadData 实现
- [DynamicButton][44]：带动画的特殊字符按钮显示，Yet another animated flat buttons in Swift 
- [ESTMusicIndicator][45]：音乐播放的动态效果控件
- [ENSwiftSideMenu][46]：一个像 Gmail 客户端那样的侧滑菜单，参考：[在 Swift 项目中实现侧滑菜单-利用 SWRevealViewController][47]
- [ElasticTransition][48]：皮筋侧滑效果
- [expanding-collection][49]：展开、查看列表项的 Peek/Pop 用法及动画效果，ExpandingCollection is a card peek/pop controller
- [Floater\_][50] ：加一个浮动指尖图标到你的演示应用，如何安装、使用步骤都写的比较清楚。Add a floating fingertip to your app demo.
- [FlowingMenu][51]：皮筋弹跳式菜单出场，Interactive view transition to display menus with flowing and bouncing effects in Swift
- [folding-cell][52]：自然流畅、清新的单元格折叠动画类及演示，FoldingCell is an expanding content cell inspired by folding paper material
- [greedo-layout-for-ios][53]：响应式网格布局，能够在固定 cell 高度的情况下自动调整图片宽度，让每行的图片都能以最佳状态显示。Full aspect ratio grid layout for iOS
- [Gecco][54]：一个高亮引导向导界面库，Simply highlight items for your tutorial walkthrough
- [Hexacon][55]：在手机上用多边形展示数据，类似 Apple Watch，A fancy hexagonal layout for displaying data like your Apple Watch
- [Iconic][56]：制作图标，Auto-generated icon font library for iOS
- [Material][57]：能很好地帮助你迅速打造一款在 iOS 上的 Material 风格应用，它提供了像是按钮、导航栏、点击的动画效果等一系列 Material 风格的 UI 元素供你使用。A beautiful graphics framework for Material Design in Swift. 
- [NextGrowingTextView][58]：自适应高度的多行文本输入框，The next in the generations of 'growing textviews' optimized for iOS 7 and above.
- [NumberMorphView][59]：可爱的数字补间（变身）动画类库，A label view for displaying numbers which can transition or animate using a technique called number tweening or number morphing
- [PNChart-Swift][60]：画图表工具，A simple and beautiful chart lib used in Piner and CoinsMan for iOS
- [PMAlertController][61]：可定制弹窗组件替代官版不可定制的 UIAlertController。PMAlertController is a great and customizable substitute to UIAlertController
- [Popover][62] ：UIPopoverController 效果的 Swift 实现，类似微信首页右上角按钮的弹出视图。Popover is an UIPopoverController like control for iOS.
- [preview-transition][63]：图片展示预览和转场，PreviewTransition is a simple preview gallery controller
- [pull-to-refresh][64]：一个非常易于开发者使用的下拉刷新和加载更多组件。通过一个UIScrollView的扩展，可以轻松为UIScrollView的所有子类添加下拉刷新功能。 如果你想定制组件的UI样式，只要实现指定的协议方法即可。
- [RangeSlider][65]：数值范围选择器
- [Render][66]：启发自 React 的纯 Swift 函数版基于 UIKit 封装类库，Swift and UIKit a la React.
- [Scrollable-GraphView][67]：灵动感十足的自适应、可定制滚动曲（折）线图表库，An adaptive scrollable graph view for iOS to visualise simple discrete datasets. 
- [SkyFloatingLabelTextField][68]：带图标显示浮动标签组件库，A beautiful and flexible text field control implementation of "Float Label Pattern". Written in Swift. As used in the Skyscanner TravelPro iOS app.
- [Smile-Lock][69]：可定制密码锁组件，A library for make a beautiful Passcode Lock View
- [SpreadButton][70]：一个当你点击它后会像花或者镰刀一样展开的按钮
- [SubtleVolume][71]：更隐蔽的系统音量替代指示器，Replace the system volume popup with a more subtle indicator.
- [SwipeViewController][72]：页面滑动和标签选项卡，SwipeViewController is a Swift modification of RKSwipeBetweenViewControllers - navigate between pages / ViewControllers
- [SwiftSVG][73]：实现高性能 SVG 解析，并且支持多种输入格式。A single pass SVG parser with multiple interface options (String, NS/UIBezierPath, CAShapeLayer, and NS/UIView)
- [StackViewController][74]：方便 iOS 开发者使用 UIStackView 构建表单或其它静态内容视图 。A controller that uses a UIStackView and view controller composition to display content in a list
- [Toast-Swift][75]：高可定制易用的 Toast 弹出通知组件，A Swift extension that adds toast notifications to the UIView object class.
- [TabDrawer][76]：可定制的TabBar，TabDrawer is a customizable TabBar UI element that allows you to run a block of code upon TabBarItem selection, or display a customizable drawer. 
- [Whisper][77]：用三种不同的样式来展示应用内通知，给用户带来一些新意。Break the silence of your UI, whispering, shouting or whistling at it. 
- [WaveLoadingView][78]：波浪进度指示器，参考：[就叫波浪进度指示器吧WaveLoadingIndicator][79]
- [YPDropMenuViewDemo][80]：swift实现的一款DropMenu,menuView,类似之前美团的下拉菜单，支持swift2.2，因为用到iOS8.的毛玻璃。所以想支持到iOS8.0,支持自定义，支持等宽排列，支持从左向右排列
- [ZYThumbnailTableView][81]：可展开型预览TableView，a TableView have thumbnail cell only, and you can use gesture let it expands other expansionView, all diy。参考：[ZYThumbnailTableView---堪比一个小型阅读App][82]

## 图片显示及处理
- [Blurable][83]：通过协议扩展高斯模糊任意 UIView，Apply a Gaussian Blur to any UIView with Swift Protocol Extensions
- [CartoonEyes][84]：前置摄像头捕获图像后，采用 Core Image 脸部识别 CIDetector 和漫画效果滤镜复合出卡通效果眼睛。Composite Cartoon Eyes over Face from Front Camera with CoreImage
- [Fusuma][85]：类似Instagram风格的图片浏览及相机拍照功能库，Instagram-like photo browser and a camera feature with a few line of code in Swift.
- [Gifu][86]：GIF库，High-performance animated GIF support for iOS in Swift
- [GPUImage2][87]：一个GPU加速视频和图像处理Swift框架。GPUImage 2 is a BSD-licensed Swift framework for GPU-accelerated video and image processing.
- [ImageViewer][88]：图片全屏预览组件及类库
- [MetalAcc][89]：基于GPU加速图片处理，包括高性能的滤镜。GPU-based media processing library using Metal written in Swift
- [PHImageKit][90]：图像下载&缓存，以及GIF显示库。Simple yet powerful image downloading, caching and GIF playback framework.
- [SKPhotoBrowser][91]：图片浏览工具，支持多种显示模式，支持缓存图片，支持缩放。Simple PhotoBrowser/Viewer inspired by facebook, twitter photo browsers written by swift
- [UIImageView-BetterFace-Swift][92]：更好显示人脸的缩略图工具

## 颜色相关

- [Hue][94]：16 进制转 UIColor、判断颜色是深是浅、添加 Alpha 通道、设置渐变色、计算图片主要颜色……一切你能想到的颜色操作，Hue is the all-in-one coloring utility that you'll ever need. 
- [imageColors][95]：提取图片主色、次色、景物色、背景色及所有涉及颜色工具库，Get the colours that matter
- [SwiftTheme][96]：Powerful theme/skin manager for iOS 7+ 主题/换肤, 夜间模式
